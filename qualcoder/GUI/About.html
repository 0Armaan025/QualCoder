Other details

The qda data folder contains folders for imported documents, images, audio and video. It also contains the sqlite database, named data.qda, to store coding data.

QualCoder creates a .qualcoder folder inside your home directory. This contains QualCoder.log, config.ini (for settings) and recent_project.txt (for automatic loading of the most recent project). The config file contains the name of the current coder, a default working directory and choice of font.

QualCoder is written in python 3 using Qt5 for the graphical interface.
